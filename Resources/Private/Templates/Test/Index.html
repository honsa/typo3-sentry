Sentry Test

Use the buttons below to trigger test events. For quick testing from the browser, use these buttons. For scripted tests use the curl examples shown below.

curl examples

Run these on the server (or adapt the Host header) to trigger actions without using the browser.

curl -X POST -H "Host: test.internal" \
  -d "tx_sentry_test[action]=triggerWarning&tx_sentry_test[controller]=Test" \
  http://127.0.0.1/sentry?no_cache=1

curl -X POST -H "Host: test.internal" \
  -d "tx_sentry_test[action]=triggerException&tx_sentry_test[controller]=Test" \
  http://127.0.0.1/sentry?no_cache=1
